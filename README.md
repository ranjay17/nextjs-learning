 # Features:

   - Authentication (Login / Logout)

   - JWT token stored in HTTP-only cookies

   - Protected Routes

   - /products and /products/[id] secured using Next.js middleware

  # Products Listing

   - Static list & dynamic product details

   - Dynamic Metadata

   - SEO-friendly titles using generateMetadata

   - Server Components + Client Components

   - Simple UI with inline styles

   - Image optimization using next/image

 # Tech Stack

   - Next.js 

   - React

   - JWT (jose library)

   - Middleware

   - DummyJSON API

   - Cookies (Server Side)
